# board2
Spring Boot + MyBatis를 활용한 게시판 Api 구현 연습

## 소개
Spring Boot로 프로젝트 관련 기본 설정 및 MVC 패턴을 좀 더 명확히 이해하기 위해 만들었습니다.  
인프런의 [자바 스프링부트 활용 웹개발 실무용](https://www.inflearn.com/course/backend-%EC%86%A1%EC%9E%90%EB%B0%94) 강의를 참고했습니다.

## 사용 기술

- Spring Boot 2.5.6
- Gradle
- MyBatis
- Java 11
- Oracle 11
- Bootstrap 5.0

## 주요 기능
- 게시글 조회
- 게시글 삽입
- 게시글 수정
- 게시글 삭제
- 파일 업로드
